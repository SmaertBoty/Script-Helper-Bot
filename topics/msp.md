Minescript Plus is a general purpose library created by `Razr` to simplify several functions of minescript.
                       
Minescript Plus [Main Branch](<https://github.com/R4z0rX/minescript-scripts/tree/main/Minescript-Plus>) (minescript v5.0 or higher)
                       
Minescript Plus [Legacy Branch](<https://github.com/R4z0rX/minescript-scripts/tree/legacy/Minescript-Plus>) (minescript v4.0)
                       
If you choose to use the legacy branch, you must install the `lib_java` module from the [minescript website.](<https://minescript.net/downloads>)
